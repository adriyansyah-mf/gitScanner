# GitScanner

Mass GitScanner 

## Installation

Linux

```bash
go get https://github.com/adriyansyah-mf/gitScanner/
mv gitScanner /usr/bin/
```

Linux

```bash
go get https://github.com/adriyansyah-mf/
run gitScanner.exe
```
Source Code

```bash
go run main.go
```
## Usage

```golang


go run main.go

```
